# Getting Started

## What is Nadir?

Nadir is a library of utility classes that allow you to build user interfaces. Unlike large utility libraries like Tailwind, Nadir doesn't attempt to provide utilities for every possible CSS property. Instead, it provides a system of classes that allow you to build most user interfaces with a powerful and minimal system of primitives.

## Basic Usage

Nadir classes are designed to be composable and semantic. Here's a quick example of what Nadir looks like:

```html
<div class="stack-y gap-m padding-l bg-primary">
  <h1 class="font-display font-2xl font-bold text-gray-9">Welcome</h1>
  <p class="font-base font-m text-gray-7">
    Build interfaces with minimal, powerful primitives.
  </p>
</div>
```

## Core Principles

1. **Minimal but powerful** - Only the utilities you actually need
2. **Composable primitives** - Build complex UIs from simple building blocks
3. **Predictable naming** - Consistent patterns make classes easy to remember
4. **Performance focused** - Built with Rust for fast compilation
